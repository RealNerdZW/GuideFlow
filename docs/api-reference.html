API Reference

This reference details the Guideflow API endpoints, including parameters and responses for all scenarios.

Endpoints

Endpoint Method Description
/api/guides GET Retrieve all guides
/api/guides/{id} POST Create or update a guide
/api/steps GET Get steps for a guide
/api/export POST Export guide in JSON/PDF

Example Request

fetch('/api/guides', {
    method: 'GET',
    headers: { 'Authorization': 'Bearer token' }
})
.then(response => response.json())
.then(data => console.log(data));

All scenarios, such as conditional rendering and error handling, are covered in the API.
