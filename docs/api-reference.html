API Reference

Complete reference for all GuideFlow.js packages.

@guideflow/core

ExportDescription
createGuideFlow(config)Create a GuideFlow instance
FlowDefinitionFSM-based flow definition type
StepStep configuration type
TourEngineTour lifecycle management class
FlowMachineFinite state machine internals
ProgressStorePersistence / progress tracking API

createGuideFlow(config)

import { createGuideFlow } from '@guideflow/core';

const gf = createGuideFlow({
  theme: 'minimal',       // 'minimal' | 'bold' | 'enterprise' | 'brutalist'
  debug: false,
  persistence: 'localStorage',
});

Instance Methods

MethodDescription
start(flow, context?)Start a flow by definition or ID
next()Advance to the next step
prev()Go to the previous step
goTo(stepId)Jump to a step by ID
send(event)Send a state machine event
stop()End the active tour
on(event, listener)Subscribe to tour lifecycle events
