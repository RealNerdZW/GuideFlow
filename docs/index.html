What is GuideFlow.js?

GuideFlow.js is a modular, AI-capable product tour library. It uses a finite-state machine (FSM) engine to drive multi-step guided tours, popovers, and spotlight overlays across React, Vue, Svelte, or plain JavaScript apps.

⚡ Zero Dependencies

The core engine ships with no runtime dependencies. Tree-shakeable ESM + CJS builds.

🤖 AI-Powered

Generate tours from a single prompt. Detect user intent. Answer questions about your UI. Supports OpenAI, Anthropic, and Ollama.

🔄 State-Machine Driven

Flows are defined as FSM state maps — branch, loop, and skip steps based on user actions.

📊 Built-in Analytics

First-class adapters for PostHog, Mixpanel, Amplitude, Segment, and custom webhooks.

🎨 Themeable

Ships with Minimal, Bold, Enterprise, and Brutalist themes. Fully overridable via CSS variables.

🧪 A/B Testing

Deterministic, client-side experiment engine. No server round-trip needed.

Quick Install

npm install @guideflow/core
# React
npm install @guideflow/react
# Vue
npm install @guideflow/vue
# Svelte
npm install @guideflow/svelte

Packages

  • @guideflow/core — Framework-agnostic FSM engine
  • @guideflow/react — React components and hooks
  • @guideflow/vue — Vue 3 plugin and composables
  • @guideflow/svelte — Svelte stores
  • @guideflow/ai — AI tour generation (OpenAI, Anthropic, Ollama)
  • @guideflow/analytics — Analytics and A/B testing
  • @guideflow/cli — CLI scaffolding and Studio
