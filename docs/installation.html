Installation

GuideFlow.js requires Node.js ≥ 18.

Install via npm

npm install @guideflow/core

Install via pnpm

pnpm add @guideflow/core

Install via yarn

yarn add @guideflow/core

Framework Packages

Install the package for your framework alongside core:

# React
npm install @guideflow/core @guideflow/react

# Vue 3
npm install @guideflow/core @guideflow/vue

# Svelte
npm install @guideflow/core @guideflow/svelte

Optional Packages

# AI tour generation
npm install @guideflow/ai

# Analytics & A/B testing
npm install @guideflow/analytics

# CLI tools
npm install -g @guideflow/cli

Import Styles

Import the default stylesheet once near your app entry point:

import '@guideflow/core/styles';
The styles export is available as @guideflow/core/styles. Custom themes are available via CSS variables — see the Usage guide.

CDN (UMD)

For quick prototyping, load GuideFlow directly from a CDN:

<script src="https://cdn.jsdelivr.net/npm/@guideflow/core@0.1.4/dist/index.global.js"></script>
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/@guideflow/core@0.1.4/dist/styles/index.css">

The UMD build exposes GuideFlow.createGuideFlow on the global scope.
