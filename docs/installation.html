Installation Guide

This guide walks you through installing Guideflow on various platforms, including Windows, macOS, Linux, and web-based environments.

Prerequisites

  • Node.js (version 14 or higher)
  • Git
  • A code editor (e.g., VS Code)

Installation Steps

  1. Clone the repository: git clone https://github.com/your-repo/guideflow.git
  2. Navigate to the directory: cd guideflow
  3. Install dependencies: npm install
  4. Run the application: npm start

Web Deployment

For GitHub Pages deployment:

npm run build
git add dist/
git commit -m "Deploy to GitHub Pages"
git push origin main

All scenarios, including offline installations and cloud deployments, are supported.
